# 英雄经验值公式

（AI生成，未经检验）

魔兽争霸3中，英雄在L级时升级所需的经验值计算公式为：

```scss
f(x) = A + B + (L-1) × C + D × (英雄(L-1)级时升级所需经验值)
```

其中：

* A、B为固定因素
* C为等级因素
* D为上一个值因素
* L为当前英雄等级 

这个公式在游戏的 **高级-游戏平衡性常数** 中可以找到相关参数设置，勾选 **自定义数据** 后可以查看和修改这些参数。
